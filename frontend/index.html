Errores y advertencias
    Sintaxis aceptada

    Palabras equivalentes

    • obtener / seleccionar / listar / traer → SELECT
    • desde / de / en → FROM
    • donde / cuando / con → WHERE
    • y / o → AND / OR
    • ordenar por → ORDER BY
    • ascendente / descendente → ASC / DESC
    • limite → LIMIT

    Instrucciones

    • insertar / agregar → INSERT
    • actualizar / modificar → UPDATE
    • establecer / poner → SET
    • eliminar / borrar → DELETE
    • crear tabla → CREATE TABLE

    Tipos y valores

    • entero / texto / decimal / fecha / booleano
    • primario / obligatorio / unico
    • verdadero / falso → TRUE / FALSE
    • Operadores: = != < > <= >=
